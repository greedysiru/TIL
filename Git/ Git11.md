## 1. Git, Github

### Git

* Distributed Version Control Sysytem
  * 분산형 버전관리 시스템
* 오픈소스
* 모든 변경 사항을 트래킹
  * 무엇이, 어디에서, 뭐가, 언제, 누가
  * history



### Github

* 깃 파일들을 업로드 하는 곳(온라인)
  * 또는 공유



## 2. Github Desktop

* git을 위한 make up
