# Array vs. LinkedList

| 경우             | Array                                   | LinkedList                   |
| ---------------- | --------------------------------------- | ---------------------------- |
| 특정 원소 조회   | O(1)                                    | O(N)                         |
| 중간에 삽입 삭제 | O(N)                                    | O(1)                         |
| 데이터 추가      | 데이터 추가시 새로운 메모리 공간을 할당 | 맨 뒤의 노드만 동적으로 추가 |
| 정리             | 데이터에 접근하는 경우가 빈번한 경우    | 삽입과 삭제가 빈번한 경우    |
