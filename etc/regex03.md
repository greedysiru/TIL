# 정규표현식(3)

**본 내용은 드림코딩 by 엘리님의 정규표현식, 더이상 미루지 말자 강의를 토대로 작성하였습니다.**



## 정규표현식 문자 관련 문법

```
\ 특수 문자가 아닌 문자
. 어떤 글자 (줄바꿈 문자 제외)
\d digit 숫자
\D digit 숫자 아님
\w word 문자
\W word 문자 아님
\s space 공백
\S space 공백 아님
```
