## 2.1024

단어 1개 입력받아 나누어 출력하기

```python
x = input();
for i in x:
  print("'%c'" %i)
```

%c는 한글자씩 출력



## 3. 1025

정수 1개 입력받아 나누어 출력하기

```python
print("["+str(int(n[0])*10000) +”]”);
print("["+str(int(n[1])*1000) +”]”);
print("["+str(int(n[2])*100) +”]”);
print("["+str(int(n[3])*10) +”]”);
print("["+str(int(n[4])*1) +”]”);
```

## 5. 1027
