

# [프로그래머스] 정수 찾기 - JavaScript 풀이

**본 게시물은 프로그래머스의 연습 문제 풀이입니다. 저작권은 (주) 그랩에게 있습니다**

## 자바스크립트 코드

```JavaScript
const solution = (num_list, n) => num_list.includes(n)? 1 : 0;
```

배열 빌트인 API인 includes는 전달받은 것이 요소로 존재 여부를 boolean으로 return 한다.
