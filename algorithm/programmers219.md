

# [프로그래머스] 배열의 길이에 따라 다른 연산하기

**본 게시물은 프로그래머스의 연습 문제 풀이입니다. 저작권은 (주) 그랩에게 있습니다**

## 자바스크립트 코드



```JavaScript
const 짝수인지 = (숫자) => 숫자 % 2 === 0;

const solution = (배열, 정수) => 짝수인지(배열.length)? 배열.map((원소, 인덱스) => 짝수인지(인덱스)? 원소 : 원소 + 정수) : 배열.map((원소, 인덱스) => 짝수인지(인덱스)? 원소 + 정수 : 원소); 
```
