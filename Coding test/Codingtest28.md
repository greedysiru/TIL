# 1. 이분검색

## 문제 풀이

```python
n, m = map(int, input().split());
arr = list(map(int, input().split()));
s = 0; # 시작점
e = n; # 끝점


arr.sort(); # 오름차순 정렬

while s <= e: # 이분 탐색
    mid = (s + e) // 2; # 중간점
    if arr[mid] == m: # 목표값 찾은 경우 탐색 완료
        print(mid + 1);
        break;
    elif arr[mid] < m : # 중간점보다 목표값이 큰 경우
        s = mid + 1;
    elif arr[mid] > m : # 중간점보다 목표값이 작은 경우   
        e = mid - 1;
```

# 2. 랜선 자르기 (결정 알고리즘)

## 문제 해설

* 이분검색 : 특정 범위 내에 답이 있는 것을 알 수 있는 경우

  * 좋은 답을 좁히며 찾아간다.

  

## 문제 풀이

```python
import sys
sys.stdin = open("in1.txt", "rt")
k, n = map(int, input().split());
arr = [0]* k;
for i in range(k):
    arr[i] = int(input());

arr.sort(reverse=True); # 내림차순 정렬
s = 0; # 시작점
e = arr[0]; # 끝점

while s <= e: # 이진 탐색
    mid = (s + e) //2; # 중간점
    cnt = 0; # 카운트
    for x in arr: # 나눈 값을 카운트
        cnt += x // mid;
        print(mid)
    if cnt == n:       # 목표값을 찾으면 출력 후 중단
        print(mid);
        break
    elif cnt < n:
        e = mid - 1;
    elif cnt > n:
        s = mid + 1;
```

## 문제 풀이

    D = 0; # DVD

## 문제 답안

      cnt = 1;
      sum = 0;
      for x in Music:
          if sum + x > capacity:
              cnt += 1
              sum = x;
          else:
              sum += x;

maxx = max(Music);
lt = 1;
rt = sum(Music);
res = 0
while lt <= rt:  # 이분 검색
    mid = (lt + rt) // 2;
